Platform capabilities

Technical overview of core functionality and export options.

  • CF
    Core functionality
    Real-time calculation engine with deterministic outputs.
  • UD
    User dashboard
    Customizable views, activity logs, and quick access to recent results.
  • DX
    Data export
    CSV and JSON export with metadata and timestamps for audits.
  • RT
    Real-time analytics
    Stream metrics to monitor throughput and latency.
  • AL
    Audit logs
    Immutable records for calculation runs and exports.
  • RB
    Role-based access
    Granular permissions for team and API access control.
