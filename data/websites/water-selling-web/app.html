Platform capabilities

Water Selling Web combines route management, payments, and reporting in one seller-friendly app. Below are core features that power your day-to-day.

  • 1
    Order & route management
    Assign deliveries, prioritize urgent orders, and view route progress in real time.
  • 2
    User dashboard insights
    A centralized user dashboard surfaces daily volume, fulfillment times, and customer histories.
  • 3
    Secure payments & reconciliation
    Track payments per order and reconcile payouts with simple tools built for sellers.
  • 4
    Data export & reporting
    Export transaction logs, daily sales, and inventory snapshots for accounting or analysis.
  • 5
    API integration
    Connect your sales systems or POS via API to push orders and pull daily metrics programmatically.

Connect your API

Integrate with your systems by pasting a secure token. Tokens are scoped to read orders, submit deliveries, and trigger exports. Generate tokens in Settings if you haven't created one yet.

Once connected, you can programmatically request data export files from the user dashboard and automate regular data export routines.
Tip: Use the "data export" feature in the user dashboard to download CSVs for accounting or to integrate with external reporting tools.
